3Frontend Deployment (Vercel)

Deploy your frontend to Vercel:

  • Go to Vercel.com and sign up
  • Click "New Project"
  • Import your GitHub repository
  • Framework: Vite
  • Build Command: npm run build
  • Output Directory: dist
  • Add environment variable: VITE_API_URL
